敬邀參與107年10月29日
【AWS雲創學院成立暨AWS研習營】

輔仁大學即將攜手AWS亞馬遜公司成立「雲創學院」，引入最高端的雲端技術，培育一流的雲端產業人才。誠摯地邀請您參與本活動，藉由產、官、學界的經驗分享與交流，帶您瞭解雲創時代的發展新契機！凡是全程參與本活動的師生，可獲得AWS官方認證之結業證書及精美小禮物，還可藉由參與AWS Educate program獲得最高美金$200元的雲端積分，立即來展開雲端學習之旅吧！

  • 一、活動日期：
    107年10月29日上午場：9:30~12:00、下午場：13:30~16:30(可擇一參與)
  • 二、活動地點：
    國璽樓一樓國際會議廳 (MD152)
  • 三、主辦單位：輔仁大學、AWS亞馬遜公司
  • 四、指導單位：新北市政府
  • 五、活動議程：
    上午場-AWS雲創學院揭牌儀式及專題演講
    開 幕：雲創學院揭牌儀式
    講題一：輔仁大學AWS雲創學院願景介紹
    講題二：教育 x 企業需求 x AWS User Group Taiwanbr
    講題三：如何透過 AWS 創造企業新藍海
    (上午場敬備午餐)

    下午場-AWS研習營
    講題一：雲端創新的新標準
    講題二：從教室上雲端之實踐經驗談
    講題三：AWS銜接校園與職場新橋樑
  • 六、教育訓練時數申請中
  • 七、活動代碼：26912
  • 八、報名網址：http://activity.dsa.fju.edu.tw/Activity.jsp?activityID=26912
